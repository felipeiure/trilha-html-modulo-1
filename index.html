História do HTML

HTML foi inventado por um físico chamado Tim Berners-Lee, e teve como objetivo facilitar a comunicação e compartilhamento de documentos entre pesquisadores. Ele iniciou o projeto em 1980 e só teve a primeira página HTML publicada em 1991. Ele também criou o protocolo HTTP e o primeiro navegador chamado World Wide Web, que se tornou o famoso WWW (Rede Mundial de Computadores).

Ao longo dos anos o HTML vem evoluindo e hoje estamos na versão 5, que foi desenvolvida para aperfeiçoar a experiência da World Wide Web para os desenvolvedores e usuários finais.

Estrutura básica do HTML

O HTML tem basicamente três partes, a primeira é a tag <html> que contém todas as demais, ela é a raiz do documento HTML.

Logo em seguida vem a tag <head> que é o cabeçalho do documento html, nela são dadas informações gerais, como título, folhas de estilo, scripts e palavras-chaves.

Por último a tag <body> que é o corpo da página HTML, é onde o contéudo será mostrado no navegador para o usúario, sejam imagens, vídeos, textos, etc

Entendendo a diferença entre listas ordenadas e não ordenadas

Listas Ordenadas é uma lista de itens númerados. Para criar listas ordenadas usamos a tag <ol> seguido da tag <li>.

Exemplo do código:
<ol>
<li>José</li>
<li>Ricardo</li>
</ol>
Resultado no navegador:
  1. José
  2. Ricardo

Listas Não Ordenadas é uma lista de itens não numerados, normalmente listados com círculos pretos. Nele usamos a tag <ul> seguido da tag <li>.

Exemplo do código:
<ul>
<li>Arroz</li>
<li>Feijão</li>
</ul>
Resultado no navegador:
  • Arroz
  • Feijão
